System layer · buttons & the flat Nagai shadow

info success warning

Shadows are flat, hard-edged, offset blocks of --shadow — no blur. Press a button to see the offset collapse. This is Nagai's painted wall-shadow, used as the app's one elevation rule. Primary CTAs use --brand (blue) — never coral, because a red-family primary reads as destructive. Error/destructive states use coral with text + icon disambiguating context — no separate --danger token for now (see brainstorm).

Accent layer · where coral is right

Coral (--accent) is for look here, never do this. Live indicators, hot states, attention badges, highlight callouts, the wordmark accent square. Used sparingly — the lone red car in the composition.

Atmosphere layer · banded gradient sky (daytime register)

Gradients are banded — discrete hard-edged steps, not a smooth blend. This is the "feels like pixel art" warmth. Lives in heroes / loading screens / empty states, never on buttons.

My read: sunset with sun disc is the strongest — more memorable than daytime (more chromatic drama, more "logo-like"), and the sun reads as a focal point that completes the "sun setting over the horizon" metaphor without anyone having to be told. Daytime is more refined but less distinctive — sky gradients alone are common enough that a sky-only mark could read generic. The sun disc survives down to 16px legibly. Pick whichever lands.

Canvas drawing palette · current vs. proposed direction

A different artifact from the UI palette above. Its job: let bots paint arbitrary, legible images from a tiny color count. Needs proper value ramps, hue-shifting, neutrals, full hue coverage — a craft, not a brand set. The brainstorm's decision: adopt the professionally-honed Endesga (EDG) family.

Note — EDG8/16/32 are not strict supersets of each other. They're an aesthetically coherent family by the same artist, but a bot painting in EDG8 colors would not have valid indices on an EDG32 canvas without remapping. Whether the 8/16/32 tiers need nesting is still open — deferred to the tier-mechanic milestone, possibly with a designer partnership.
